About This Project

Creator: Dylan Miller

"Imagine" serves as a visualization tool, enabling users to imagine and immerse themselves in their perfect day. While creating a fun and interactive experience was essential, my primary goal with this project was to gain hands-on experience with various software and technologies, including Flask, Terraform, AWS, Docker, and more. Aspiring to be a proficient DevOps engineer, projects like this have been invaluable in deepening my understanding of key technologies and honing my ability to bring ideas to life.

Flask

Utilized as the web framework to build the website and handle user interactions for inputting keywords and displaying the generated "day in the life" scenarios from the ChatGPT API.

AWS

Provisioned a Linux AMI EC2 instance using Terraform to host the Flask application, offering a scalable and accessible infrastructure.

Terraform

Used to define the infrastructure as code, automating the setup of the AWS EC2 instance, ensuring reproducibility and consistency in the deployment process.

Docker

Containerized the Flask application along with its dependencies, ensuring consistent deployment across different environments.

SQLite

Employed as the lightweight relational database management system to store user responses, facilitating public sharing of "day in the life" scenarios on the site.

Jinja

Integrated as the templating engine within the Flask application allowing for dynamic content rendering.

GitHub

Utilized as the version control system to host the code repository, enabling efficient collaboration, version tracking, and integration with a future CI/CD pipeline.

ChatGPT API

Integrated into the Flask application, enabling users to input keywords and receive customized responses, forming their personalized "day in the life" scenarios.
